## 20:50 more complete responsive facebook-like page layout ``` /* desktop */ .container { display: grid; grid-template-areas: "head head" "nav subnav" "nav main" "foot foot"; grid-template-columns: 5em 1fr; grid-template-rows: 3em auto 1fr auto; } /* mobile*/ @media (max-width: 500px) { .container { grid-template-areas: "head" "nav" "subnav" "main" "foot"; grid-template-columns: 1fr; grid-template-rows: 3em auto auto minmax(2em, 1fr) auto; } } /* common */ .hero{ grid-area: head; } .nav{ grid-area: nav; } .subnav{ grid-area: subnav; } .content{ grid-area: main; } .footer{ grid-area: foot; } ```
